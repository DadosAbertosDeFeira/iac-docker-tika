# iac-docker-tika

Requisitos
----------
Informar a versão do TIKA Server através do ARG TIKA_VERSION 

Necessário ter cadastrados no secrets:


 | Var | Desc |
 |:----- |-----|
 | DOCKERHUB_USERNAME  | Usuario do Docker |
 | DOCKERHUB_TOKEN  | Token do Usuario |
